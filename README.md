## Test Script

A test scripting language for learning purposes.  Can evaluate int and float math operations as of now.  Contains a lexer, parser, and interpreter.

### Examples

**Basic arithmetic**

```c
2.6 + 2.6
// 5.2
```

```c
20 - 4
// 16
```

```c
3 * 4
// 12
```

```c
25 / 5
// 5
```

```c
5 ^ 3
// 125
```

```c
-5
// -5
```

```c
--5
// 5
```

**Order of Operations**

```c
4 + 3 * 10
// 34
```

```c
(4 + 3) * 10
// 70
```
